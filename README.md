# Pixel Art Editor

A lightweight **Pixel Art Editor** built with **Vite + React 19**.  
Draw pixel-perfect art with layers, minimap, and undo/redo functionality.

---

## Features

- Pixel-perfect drawing with **pen**, **eraser**, **fill**, **line**, **rectangle**, and **circle** tools
- **Layer management** with visibility & opacity
- **Minimap** preview
- Undo / Redo history
- Zoom and resize canvas
- Export canvas as image

---

## Tech Stack

- **React 19** + **Vite 8**
- **ESLint** for linting
- **MUI** for UI components
